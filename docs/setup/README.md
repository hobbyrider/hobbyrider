# Setup & Configuration

Setup instructions and configuration guides for Hobbyrider.

## Available Guides

### Core Setup
- **`DEPLOYMENT.md`** - Production deployment guide
- **`DEVELOPMENT.md`** - Development environment setup

### Services & Integrations
- **`AUTH_SETUP.md`** - Authentication setup (OAuth, Email)
- **`RESEND_SETUP.md`** - Email service (Resend) configuration

### Checklists
- **`PRODUCTION_DEPLOYMENT_CHECKLIST.md`** - Production deployment checklist

## Quick Start

1. **Development**: Start with `DEVELOPMENT.md`
2. **Production**: Follow `DEPLOYMENT.md`
3. **Checklist**: Use `PRODUCTION_DEPLOYMENT_CHECKLIST.md` before deploying

## Adding New Setup Guides

When documenting new setup procedures:
1. Place them in this `docs/setup/` directory
2. Use clear, step-by-step instructions
3. Include environment variables needed
4. Update this README
