# Guides

User and admin guides for using and managing Hobbyrider.

## Available Guides

- **`ADMIN_QUICK_REFERENCE.md`** - Quick reference for admin features
- **`MODERATION_GUIDE.md`** - Content moderation guide

## Adding New Guides

When creating new guides:
1. Place them in this `docs/guides/` directory
2. Use descriptive filenames (kebab-case)
3. Update this README with a brief description
4. Follow the project's documentation style
